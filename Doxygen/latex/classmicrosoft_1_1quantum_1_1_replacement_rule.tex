\doxysubsection{Detailed Description}
Rule that describes a pattern and how to make a replacement of the matched values. The class contains a Operand\+Prototype which is used to test whether an LLVM IR value follows a specific pattern. The class also holds a function pointer to logic that allows replacement of the specified value.
