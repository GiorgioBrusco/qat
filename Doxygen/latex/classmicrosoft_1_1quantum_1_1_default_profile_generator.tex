\doxysubsection{Detailed Description}
\mbox{\hyperlink{classmicrosoft_1_1quantum_1_1_default_profile_generator}{Default\+Profile\+Generator}} defines a profile that configures the rule set used by the \mbox{\hyperlink{classmicrosoft_1_1quantum_1_1_profile}{Profile}} pass. This profile is useful for generating dynamic profiles and is well suited for testing purposes or YAML configured transformation of the IR.
