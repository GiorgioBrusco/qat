\doxysubsection{Detailed Description}
\mbox{\hyperlink{classmicrosoft_1_1quantum_1_1_configuration_manager}{Configuration\+Manager}} is a class that holds a collection of configurations (sections). Each of these sections are embodied in their own class with a one-\/to-\/one mapping between configuration section and the configuration type. As an example, if one wishes to make a configuration for the class Foo, one would create a class Foo\+Config which would hold all the variables that are configurable and then add Foo\+Config to the \mbox{\hyperlink{classmicrosoft_1_1quantum_1_1_configuration_manager}{Configuration\+Manager}} using {\ttfamily \mbox{\hyperlink{classmicrosoft_1_1quantum_1_1_configuration_manager_a26df4c5ae50f18bac796518303f8aa92}{add\+Config()}}}. For Foo\+Config to fulfill the concept of a configuration, it must implement a setup functions whose first argument is the \mbox{\hyperlink{classmicrosoft_1_1quantum_1_1_configuration_manager}{Configuration\+Manager}}.
