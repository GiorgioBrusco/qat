\doxysubsection{Detailed Description}
Helper class to enable literals for IR patterns. The main purpose of this class is to enable notation that allows one write {\ttfamily \char`\"{}name\char`\"{}\+\_\+cap = operand\+Generator()} where the operand generator is a function which creates a IOperand\+Prototype\+Ptr. This notation means that whenever a operand is matched, the matched value is stored under \char`\"{}name\char`\"{}.
