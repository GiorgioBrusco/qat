\doxysubsection{Detailed Description}
\mbox{\hyperlink{classmicrosoft_1_1quantum_1_1_profile}{Profile}} class that defines a set of rules which constitutes the profile definition. Each of the rules can be used to transform a generic QIR and/or validate that the QIR is compliant with said rule.
